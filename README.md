# SWEA Header :: swheader

SWEA 문제풀이인을 위한 SWEA Header Generator


## 기능

더 이상의 `import sys`는 없다.

![command](https://media.giphy.com/media/lqFYOssHNkCfgUvtG5/giphy.gif)


## 사용법

1. 파이썬(Python) 파일 생성 

2. `swea` 입력 후, **tab**

3. 문제명 입력 후, **tab**

4. test 파일명 확인 후, **tab**

5. 문제풀이
